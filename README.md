# detect-bubble
Classify & detect dialogue bubbles

## requirements
- opencv-python
- selectivesearch
- tensorflow2

## Dataset
Contains 3 labels: 
- Cloud
- Oval
- Square

Link : https://drive.google.com/drive/folders/15hcGAZtAKd9UMmt-kLcD-3nT99uLC7Qv?usp=sharing

Credits for this dataset to https://github.com/Shiroechi
